Database

The Database class represents a file-based database. It stores tables and relationships in files, and uses the JSON, YAML and XML formats to define data. It has the following properties and methods:

Properties

  • tables: A list of table names, stored as filenames in the format {tablename}.json
  • relationships: A list of relationships between tables, stored as strings in the format {table1}:{table2}
  • declarations: A list of table declarations, stored as filenames in the format {tablename}.yaml
  • name: The name of the database
  • description: A description of the database

Methods

  • add_table(name): Adds a new table to the database with the given name
  • delete_table(name): Deletes the table with the given name from the database
  • add_relationship(table1, table2): Adds a relationship between the two given tables
  • delete_relationship(table1, table2): Deletes the relationship between the two given tables
